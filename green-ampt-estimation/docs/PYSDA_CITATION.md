# Green-Ampt Estimation with PySDA

This project utilizes the `pysda` library to fetch soil data from the NRCS Soil Data Access (SDA) service.

## Citation and Acknowledgment

The `pysda` library is an open-source project developed by the National Cooperative Soil Survey (NCSS) community. We gratefully acknowledge their work, which provides the critical data access functionality for this toolkit.
